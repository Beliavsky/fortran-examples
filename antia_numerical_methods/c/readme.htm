NUMERICAL  METHODS  FOR  SCIENTISTS  AND ENGINEERS

C  PROGRAMS



This directory contains C programs and corresponding outputs. All programs use double precision (type double) floating point arithmetic.

These files are arranged in following 2 subdirectories:
 


 fun       222 C functions described in Appendix C
 
 xmp      118 programs illustrating usage of functions described in Appendix C. Sample outputs from each of these programs are also included.
