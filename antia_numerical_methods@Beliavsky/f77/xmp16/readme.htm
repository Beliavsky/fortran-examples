FORTRAN PROGRAMS


This directory contains Fortran 77 programs using various subprograms described in Appendix B. All programs use quad precision (REAL*16 / COMPLEX*32) arithmetic. Single and double precision versions of these programs can be found in directories f77/xmp4 and f77/xmp8 respectively. The subprograms can be found in the directory f77/sub16

All these programs have been tested to some extent, but there may be some bugs. Users are welcome to use these programs at their own risk. Any bugs discovered in these routines may be notified to

antia@tifr.res.in

The main purpose of the programs in this directory is to provide concrete examples of using the subprograms described in Appendix B. Most of these programs are based on either solved examples or exercises in the book. Sample output obtained from these programs is also included in files *.out. These output files may be useful to check if the program is running correctly (at least giving the same output!) on your computer. Because of differences in roundoff errors between different compilers and computers the output may not exactly match on two different machines. In most cases the output refers to output on Fortran logical unit 6 which is normally the terminal. This output has been redirected to a file which is included in the directory. In a few cases (anneal.f, bsplin.f, rls.f, smooth.f) the programs directly write to a file on the disk. Of these bsplin.f and smooth.f do not have any significant output on the screen and it is not included in the online material. While for anneal.f and rls.f the screen output is included in addition to the files directly written. The input file kernel containing kernels for inverse problem (Example 13.5) as required by rls.f is kept in the base directory to avoid duplication.

Because of limitations imposed by precision of arithmetic some of the outputs in this directory do not correspond to the same problem as in the double precision version. In most cases, the effect of precision of arithmetic can be gauged by comparing the outputs in this directory with those in f77/xmp8. Because of limitations mentioned in f77/sub16/readme all programs in this directory may not give the full accuracy expected from quadruple precision arithmetic. Some of these programs may require very long time to complete execution.

Apart from 123 pairs of Fortran programs and the corresponding output files this directory contains the following files:

 readme.htm   This file
 xmp.ind   Index of subprograms used in different examples

 xmplsta.htm   Alphabetic list of subprograms with HTML links
